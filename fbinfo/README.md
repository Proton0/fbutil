
# fbinfo

A simple python command that gets the framebuffer  information using adb 
## Run Locally

Clone the project

```bash
  git clone https://github.com/proton0/fbutil
```

Go to the project directory

```bash
  cd fbutil
```

Install dependencies

```bash
  pip3 install -r requirements.txt
```

Run the tool

```bash
  cd fbinfo
  python3 main.py
```
